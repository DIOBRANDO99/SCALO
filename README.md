# SCALO
SCALO è una piattaforma web per cercare voli tra un aeroporto di partenza e uno di arrivo con scalo obbligato (es. “passare da Istanbul”). L’utente inserisce codici IATA, date e scalo desiderato; il sistema interroga le API, filtra le soluzioni che lo includono e mostra prezzi, durata totale, tempo di stopover e compagnie aeree.
